        <p>Veamos cada una de estas partes en detalle:</p>
164
        <ol>
165
          <li><b>Inicialización:</b> Se define una variable y se le asigna un valor inicial. Esta parte del bucle se ejecuta solo una vez, al comienzo de la iteración.</li>
166
          <li><b>Condición:</b> Se establece una condición que se evalúa al principio de cada iteración del bucle. Si la condición se cumple, se ejecuta el código dentro del bucle. Si la condición no se cumple, el bucle se detiene y la ejecución continúa después del bucle.</li>
167
          <li><b>Expresión de iteración:</b> és de cada iteración del bucle, se evalúa esta expresión. Por lo general, se usa para actualizar la variable de inicialización. Esta expresión puede omitirse si no es necesaria.</li>
168
          <li><b>Código a ejecutar en cada iteración:</b> Es el código que se ejecuta en cada iteración del bucle.</li>
